3D-Cell-Annotator: an open-source tool for single cells segmentation/annotation in microscopy 3D images

Cancer 3D multicellular spheroid segmented in single cells

3D-CELL-ANNOTATOR, A MITK PLUGIN:
 
3D-Cell-Annotator is implemented in C++/CUDA. It is provided as a patch for the segmentation module of the Medical Interaction Toolkit (MITK), a popular freely available medical image analysis software. You can download the 3D-Cell-Annotator enabled MITK distribution at the Download Section of this Website. It requires a recent version of the NVidia drivers and a CUDA enabled GPU.
